Guidelines

  • Registered email required → Only users with an existing account can log in.
  • Password mandatory → Ensures authorized access to the account.
  • Email format validation → Prevents incorrect or invalid email entries.
  • Secure encrypted authentication → Protects user credentials during login.
  • Limited login attempts → Prevents brute-force and unauthorized access.
  • Forgot password support → Allows users to recover access securely.
  • Session-based access → Keeps users logged in safely during active use.
  • Informative placeholders → Guides users on what input is required.
  • Clear error messages → Helps users quickly correct mistakes.
